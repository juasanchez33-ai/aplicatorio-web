# Aplicativo Web para el Manejo de Finanzas Personales - Premium

Este repositorio contiene el **80% del desarrollo** de una aplicación web financiera profesional, diseñada para la gestión personal de activos, ingresos y egresos.

## 🚀 Descripción
Proyecto de gestión financiera que integra un potente backend en FastAPI con una interfaz de usuario moderna (Glassmorphism), permitiendo el seguimiento detallado de la salud económica del usuario.

## 🛠 Tecnologías Usadas
- **Backend**: FastAPI (Python)
- **Frontend**: HTML5, Vanilla CSS, JS
- **Diseño**: Tailwind CSS, Glassmorphism UI
- **Base de Datos**: SQLite / Firebase Data Connect
- **Gráficos**: ApexCharts

## 📂 Estructura del Proyecto
```text
Proyecto_Finanzas_80
├── backend/            # Lógica del servidor y rutas
├── frontend/           # Interfaz de usuario y assets
├── firebase/           # Configuración de Firebase
├── database/           # DB local y diagramas
├── mockups/            # Mockups del diseño final
└── docs/               # Documentación completa
```

## 🚥 Instalación Rápida
1. Instale dependencias: `pip install -r requirements.txt`
2. Configure el entorno: renonbre `.env.example` a `.env`
3. Inicie el sistema: `cd backend && python app.py`

## ✨ Características Principales
- **Dashboard Dinámico**: Estadísticas en tiempo real.
- **Botón FAB (+)**: Acceso rápido para registrar transacciones.
- **Gestión de Activos**: Seguimiento de BTC, Acciones y más.
- **Modo Educativo**: Módulo de aprendizaje financiero integrado.
