# Registro de Cambios - Aplicativo Web de Finanzas

Este documento detalla el progreso y las ediciones realizadas en el proyecto para asegurar transparencia y seguimiento de las mejoras implementadas.

## [v1.0.5] - 2026-03-17
### Añadido
- **Historial de Cambios (Commits)**: Se implementó este archivo `CHANGELOG.md` para documentar cada paso del desarrollo.
- **Sección de Novedades en Dashboard**: Ahora el cliente puede ver las últimas actualizaciones directamente desde el panel principal.

### Mejoras Estéticas
- **Refinamiento de Glassmorphism**: Se ajustaron los bordes de neón y el desenfoque de fondo para un diseño más moderno y sofisticado.
- **Animaciones Premium**: Se agregaron micro-transiciones a las tarjetas de balance e ingresos.

### Preparación de Despliegue
- **Entorno de Producción**: Se configuraron las variables necesarias para el despliegue seguro en la nube.
- **Seguridad**: Activación de HTTPS y validación de endpoints para el servidor de producción.

---

## [v1.0.4] - 2026-03-15
### Añadido
- **Autenticación SMTP**: Integración con Gmail para el envío de códigos OTP de seguridad.
- **Sistema de Recuperación de Contraseña**: Flujo completo para usuarios que olvidaron su clave.

---

## [v1.0.0] - 2026-03-01
### Añadido
- **Lanzamiento Inicial**: Estructura base del panel de finanzas, gestión de ingresos y egresos.
