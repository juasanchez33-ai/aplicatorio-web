# Funcionamiento del Sistema - Proyecto_Finanzas_80

Guía técnica sobre la lógica de operación del sistema financiero.

## Módulos Principales

### 1. Dashboard Operativo
- **Balance Total**: Sumatoria calculada dinámicamente de ingresos menos egresos.
- **Gráficos en Tiempo Real**: Visualización de distribución de gastos por categoría e historial semanal.

### 2. Gestión de Movimientos
- Permite registrar transacciones financieras clasificadas por tipo (Ingreso/Egreso).
- Clasificación automática por categorías (Venta, Salario, Ocio, etc.).

### 3. Seguimiento de Inversiones
- Interfaz dedicada para activos Crypto y Acciones tradicionales.
- Cálculo de rendimiento (PNL) simulado para visualización de progreso.

### 4. Control de Deudas y Pagos
- Listado interactivo de saldos pendientes.
- Recordatorios de fechas de vencimiento en el panel de servicios.

## Flujo de Usuario
1. **Autenticación**: Registro e inicio de sesión seguro.
2. **Registro**: Inserción de datos a través del FAB (+).
3. **Análisis**: Revisión de estadísticas en el Dashboard y exportación de reportes (CSV).
