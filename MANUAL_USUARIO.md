# MANUAL DE USUARIO - Aplicativo Web para el Manejo de Finanzas Personales

## 1. Bienvenida
Bienvenido a Aplicativo Web para el Manejo de Finanzas Personales, tu centro de mando financiero personal. Esta guía te ayudará a navegar por la plataforma y sacar el máximo provecho de nuestras herramientas de análisis e inversión.

## 2. Primeros Pasos
### Registro e Inicio de Sesión
- Puedes crear una cuenta usando tu correo electrónico y una contraseña segura.
- Ofrecemos integración con **Google Sign-In** para un acceso rápido y seguro.

## 3. Módulos del Sistema
### Panel de Control (Dashboard)
Aquí verás un resumen visual de tu salud financiera actual:
- **Balance Neto:** El resultado de tus ingresos menos tus gastos.
- **Gráficos de Gastos:** Visualización por categorías para identificar dónde puedes ahorrar.
- **Filtros Avanzados:** Filtra tus datos por mes, categoría o monto.

### Movimientos
Registra cada transacción para mantener tu historial al día.
- **Ingresos:** Dinero que entra (Salario, ventas, etc.).
- **Gastos:** Dinero que sale (Renta, comida, ocio).



### Gestión de Pagos
Lleva el control de tus facturas y servicios mensuales.
- Programación de pagos futuros.
- Seguimiento de estados: Pendiente, Pagado, Vencido.

### Estudio Académico
Aprende estrategias probadas para mejorar tus finanzas:
- Regla 50/30/20.
- Cálculo de Interés Compuesto.
- Guia de Finanzas Personalizada.

## 4. Preguntas Frecuentes (FAQ)
En el apartado de "Estudio Academico", haz clic en el boton "Ver Guia" para descargar tus recursos.

**¿Mis datos están seguros?**
Sí, utilizamos Firebase Auth para proteger tu sesión y una base de datos local cifrada para tus movimientos financieros personales.

## 5. Glosario
- **PnL:** Profit and Loss (Ganancias y Pérdidas).
- **Glassmorphism:** Estilo visual que simula el vidrio esmerilado.
- **Balance:** Diferencia entre activos y pasivos.
